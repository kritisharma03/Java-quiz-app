# Quiz Application (Java + PostgreSQL)

A desktop-based Quiz Application built using **Java Swing** and **PostgreSQL**.

## Features
- User login
- Multiple-choice quiz
- Questions loaded from PostgreSQL database
- Score calculation and storage
- JDBC-based database connectivity

## Technologies Used
- Java (Swing)
- PostgreSQL
- JDBC

## How to Run
1. Install Java (JDK 17+)
2. Install PostgreSQL
